## 🚀 Project Overview
The objective of this analysis is to investigate how different factors—ranging from personal background to preparation habits—impact student performance in subjects like Math, Reading, and Writing.

By leveraging data visualization and statistical analysis, this project seeks to answer whether a student’s success is purely academic or if it is significantly influenced by external variables.

📊 Key Variables Analyzed
To understand the "why" behind the scores, the project focuses on the following features:

Gender: Analyzing performance gaps between male and female students.

Race/Ethnicity: Identifying if there are disparities across different ethnic groups.

Parental Level of Education: Investigating the correlation between a parent's degree (e.g., Associate's, Master's) and the student's results.

Lunch: Using "Standard" vs. "Free/Reduced" lunch as a proxy for socio-economic status.

Test Preparation Course: Evaluating the effectiveness of completing a preparatory course before the actual exam.

🎯 Research Questions
Preparation vs. Result: Does completing a test preparation course guarantee a significantly higher score?

Socio-Economic Impact: How heavily does the "Lunch" variable (socio-economic standing) weigh on a student's ability to perform?

Parental Influence: Does a higher level of parental education lead to better student outcomes?

Subject Correlation: Is a student who excels in Reading likely to excel in Writing and Math as well?

🛠️ Methodology
Data Cleaning: Handling missing values and ensuring data types are correct for analysis.

Exploratory Data Analysis (EDA): Using Seaborn and Matplotlib to visualize distributions and correlations.

Feature Encoding: Converting categorical variables (like Gender and Ethnicity) into numerical formats for potential machine learning modeling.

Insight Generation: Drawing conclusions based on data trends to provide actionable feedback for educators.
